# AWS Polly
All packages are inside the "Installer"-folder.

A manual to setup AWS Polly can be found under "Assets/Plugins/crosstales/RTVoice/Documentation".

Please make sure you installed the correct AWSSDK-package for your "Api Compatability Level":

.NET 4.x:           AWSPolly_NET4.5.unitypackage
.NET Standard 2.0:  AWSPolly_NETStandard2.0.unitypackage

Neural voices are supported: https://docs.aws.amazon.com/polly/latest/dg/NTTS-main.html
